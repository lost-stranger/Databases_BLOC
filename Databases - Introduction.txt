2. Explain when a database would be used. Explain when a text file would be used.

A database would be used when continued access to a data set is necessary, when the dataset needs to be accessed and modified by a number of applications, where changes can be made
without locking out or overwriting the changes pushed by applications; and when multiple applications are introducing changes to the data set. 
Text files can be used to save a single instance of a data, where changes can be made to the data either one user/application at a time or by limited number of users/application.

3. Describe one difference between SQL and other programming languages.

SQL uses the '=' sign for conveying equivalence, while other languages use the sign to assign value.
SQL, being a fourth generational language, is declarative and user intuitive.

4. In your own words, explain how the pieces of a database system fit together at a high level.

Data sets in a database systems is laid out in a table format arranged in rows and columns. The columns represent the properties or information that is associated with single data objects.
The rows hold the single data object with its properties laid out in cells across the columns 

5.Explain the meaning of table, row, column, and value.

Borrowing from the above answer, a TABLE is a single data-set which hold unique data objects and their properties. Each data object occupies one full horiontal ROW, 
where each property VALUE is housed under a vertical COLUMN that hold property values of the same type. Each VALUE lies in a cell at the intersection of an unique ROW and COLUMN.
